# time_series_forecasting_with_LSTM

This project contains the notebook **`time_series_forecasting_with_LSTM.ipynb`**, which demonstrates training an LSTM model for time-series forecasting.

## Requirements

* Python 3
* TensorFlow / Keras
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

## Usage

1. Open the notebook:

   ```bash
   jupyter notebook time_series_forecasting_with_LSTM.ipynb
   ```
2. Run all cells to preprocess data, train the model, and generate forecasts.

## Description

The notebook covers:

* Data preparation
* LSTM model building
* Training and evaluation
* Forecast visualization
